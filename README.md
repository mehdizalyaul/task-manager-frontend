# Trello clone

This project is a simple Trello clone template build with React, React Context API, SCSS, and DnD Kit.

## Setup

1. Clone the repository:

```bash
git clone 
```

2. Navigate to the project directory:

```bash
cd trello-clone
```

3. Install the dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm run dev
```

5. Open your browser and go to `http://localhost:5173` to see the application in action.

## Features

- Create, edit, and delete boards
- Create, edit, and delete lists within boards
- Create, edit, and delete cards within lists
- Drag and drop cards between lists
- Responsive design
- Dark mode support

## Technologies Used

- React
- React Router
- React Context API
- SCSS
- Lucid Icons
- DnD Kit
